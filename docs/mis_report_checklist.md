## 📋 Executive Overview

This **MIS (Management Information System) Checklist** defines the key metrics, audit checks, data sources, and verification rules required for generating daily, weekly, and monthly executive MIS reports across **RetailConnect SIP**.

---

## 1. 📊 Core MIS Report Categories & Checklist Items

### A. Secondary Sales & Target Achievement MIS
- [ ] **Distributor Secondary Sales:** Total secondary sales value and volume (Cases/Units) grouped by Territory, Distributor, and Retailer Tier (A/B/C).
- [ ] **Target vs. Actual Variance:** Comparison of actual secondary sales against auto-generated targets (from Python Forecasting Engine).
- [ ] **Primary vs. Secondary Gap Analysis:** Difference between factory→distributor sales (Primary) and distributor→retailer sales (Secondary) to identify warehouse stock buildup.
- [ ] **Top & Bottom Performing SKUs:** Velocity heatmap ranking SKUs by movement, revenue contribution, and run-rate.

### B. Forecasting & Projection Engine MIS (Render Python Service)
- [ ] **Projected Demand Summary:** Next 30/60/90 day SKU-level demand predictions generated by Meta Prophet & XGBoost.
- [ ] **Forecast Accuracy Metrics (MAPE):** Mean Absolute Percentage Error tracking comparing previous predictions against actual uploaded sales.
- [ ] **AI-Generated Demand Summary (OpenRouter):** Plain-English executive summaries explaining demand spikes/drops per territory.
- [ ] **Manual Override Tracking:** Log of any manual target overrides applied by Area Sales Managers against AI-suggested targets.

### C. Field Sales Force Performance (SO MIS)
- [ ] **Beat Route Adherence:** Percentage of planned shop visits completed vs. scheduled beat routes per Sales Officer (SO).
- [ ] **Productive Call Ratio:** Ratio of total GPS-verified check-ins to actual booked orders.
- [ ] **SO Sales Leaderboard:** Ranking of SOs based on target achievement, line-item order count, and scheme conversion rate.
- [ ] **GPS Check-in Audit:** Log of offline vs. online order bookings and location verification compliance.

### D. Distributor & Inventory Health MIS
- [ ] **Distributor Stock Cover (Days of Inventory):** Available stock per SKU divided by average daily secondary sales run-rate.
- [ ] **Stockout Risk Warning:** List of SKUs near zero stock where demand is projected to be high.
- [ ] **Sales File Upload Status:** Audit check for distributor CSV/Excel upload compliance (Uploaded vs. Pending Distributors).
- [ ] **Deduplication & Validation Report:** Summary of raw upload records staged, deduplicated, and autocorrected.

### E. Schemes, Incentives & ROI MIS
- [ ] **Trade Scheme Uplift:** Sales volume increase directly attributed to active retailer schemes/offers.
- [ ] **Scheme ROI Analysis:** Scheme cost vs. incremental gross margin generated.
- [ ] **Incentive Payout Summary:** Approved and pending incentive payouts for Sales Officers and Retailers.

## 2. 🛡️ MIS Data Pipeline & Quality Control Checklist

Before finalizing any MIS Report for executive distribution, ensure all data verification checks pass:

| Check Item | Validation Rule | Responsible Layer | Status Pass Criteria |
| :--- | :--- | :--- | :--- |
| **Data Recency** | Sales data updated within last 24h | Next.js API / MongoDB | All active distributors uploaded |
| **Deduplication** | Key on `Distributor + Invoice + Date` | Distributor Upload Staging | Zero duplicate invoice records |
| **Forecasting Sync** | Render Python Service sync status | Render FastAPI `/health` | Active status (200 OK) |
| **Rollup Pre-computation** | Pre-computed summary collection refreshed | MongoDB Rollup Cron Job | Pre-computed rollups current |
| **Target Distribution** | Auto-targets deployed to SO mobile apps | Projection Engine Phase 7 | Targets approved by Area Manager |

## 3. 🗓️ MIS Reporting Cadence & Delivery Channels

```
┌──────────────────┐    ┌──────────────────┐    ┌──────────────────┐
│   DAILY MIS      │    │    WEEKLY MIS    │    │   MONTHLY MIS    │
├──────────────────┤    ├──────────────────┤    ├──────────────────┤
│ • SO Beat Count  │    │ • Forecast MAPE  │    │ • National P&L   │
│ • Daily Orders   │    │ • Scheme Uplift  │    │ • Target vs Act. │
│ • Upload Status  │    │ • Stockout Risks │    │ • SO Payouts     │
└──────────────────┘    └──────────────────┘    └──────────────────┘
```

- **Daily MIS Push (Email/App Notification):** SO adherence, daily secondary booking totals, pending file uploads.
- **Weekly MIS (Manager Dashboard):** 7-day sales velocity, stock cover days, model forecast updates.
- **Monthly Executive MIS (GM Dashboard):** National territory P&L, primary vs. secondary gap analysis, final incentive payouts.
